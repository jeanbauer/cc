Aprendendo o básico de HTML e CSS, com noções de UX e acessibilidade.

Instruções

Bem-vinda(o) ao curso de HTML e CSS.

Toda página web é escrita em uma linguagem chamada HTML. Você pode pensar em HTML como um esqueleto que dá estrutura a todas as páginas. Neste curso, vamos usar HTML para adicionar parágrafos, cabeçalhos, imagens e links a uma página.

HTML

HTML e CSS HTML significa HyperText Markup Language. Hypertext (hipertexto) significa "texto que contém links." Sempre que você clicar em uma palavra que te leve a uma nova página, você clicou em um hipertexto! Uma markup language (linguagem de marcação) é uma linguagem de programação usada para fazer com que o texto faça mais do que simplesmente ficar parado em uma página: ela pode transformar textos em imagens, links, tabelas, listas, e muito mais. HTML é a linguagem que vamos aprender. O que deixa uma página web bonita? É o CSS — Cascading Style Sheets (Folhas de Estilo em Cascata). Pense nele como a pele e a maquiagem que cobre os ossos de HTML. Vamos aprender HTML primeiro, e vamos nos preocupar com CSS nas aulas futuras. A primeira coisa que devemos fazer é configurar o esqueleto da página.

a. Sempre coloque DOCTYPE. Isso diz ao navegador que linguagem ele está lendo.

<!DOCTYPE html>

b. Sempre coloque HTML. na próxima linha. Isso dá início ao documento HTML.

<html>

c. Sempre coloque /HTML. na última linha. Isso determina o fim do documento HTML.

</html>

Terminologia básica

Para aprender mais sobre HTML, devemos aprender a falar sobre HTML. Você já nos viu usar um monte de <>s.

<>

As coisas dentro de <>s são chamadas de tags. Tags geralmente são colocadas em pares: uma tag de abertura, e uma tag de fechamento.

Exemplo de tag de abertura: 
<html>
Exemplo de tag de fechamento: 
</html>

Exemplo de uma página completa

<!DOCTYPE html>
    <html lang="en">
      <head>
        <meta charset="UTF-8">
        <title>First HTML Page</title>
        <link rel="stylesheet" href="style.css">
      </head>
      <body>
        <h1>Hey There!</h1>
      </body>
    </html>
  • Mais informações: https://developer.mozilla.org/pt-BR/docs/Web/HTML

O que é CSS

CSS (que é a sigla para Cascading Style Sheets) é a linguagem usada para descrever a aparência e a formatação do seu arquivo HTML. Uma style sheet (folha de estilo) é um arquivo que descreve como um arquivo HTML deve ser visualmente. É isso! Dizemos que essas folhas estão em modo cascading (cascata) porque as folhas podem aplicar a formatação para mais de um estilo. Por exemplo, se você disser que todos os parágrafos devem ter fonte de cor azul, mas você especifica um único parágrafo para ter fonte da cor vermelha, a linguagem CSS faz!
